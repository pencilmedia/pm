WebGain
This was a technically difficult project. Due to rollover navigation/Flash conflicts we needed to design a section of their website to look as if it were all done in flash.

The animated gif, on the left, is timed with the flash on the right. Needed to time it with loading time and keep file sizes small as possible yet keep the look corporate. Also there is a JavaScript refresh for the gif so it plays all of the value-add points.

Target audience was broadband enabled, so this helped give some leeway.
